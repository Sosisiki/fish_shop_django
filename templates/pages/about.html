О проекте

Концепция

«Магазин Рыбок» — e-commerce платформа с AI-консультантом. Решаем проблему несовместимости видов и нарушения биобаланса. ИИ анализирует параметры аквариума, подбирает совместимые виды и предупреждает о конфликтах до покупки.

Технологический стек

Backend: Django 6.0.4 (Python 3.14), архитектура MTV
Database: SQLite (с поддержкой транзакций и ORM)
AI-агент: n8n + DeepSeek V3.2 через AiTunnel
Frontend: Bootstrap 5, Vanilla JS, Fetch API
Карты: Yandex Maps JS API (геокодирование)
Безопасность: CSRF, SMS-верификация, валидация форм

Каталог и управление остатками

В базе более 50 видов пресноводных и морских обитателей. Каждая карточка содержит:

  • Рекомендуемый объём аквариума (л)
  • Уровень сложности (Новичок / Средний / Профи)
  • Характер и совместимость с другими видами
  • Реальные остатки на складе с блокировкой при stock ≤ 0

Система автоматически сортирует каталог: сначала доступные товары, затем «Распродано» внизу страницы.

Безопасность и верификация

Для защиты от ботов и обеспечения прозрачности сделок реализована обязательная двухфакторная регистрация:

  1. Ввод email, логина и номера телефона при регистрации
  2. Генерация 6-значного кода с ограничением в 5 минут
  3. Middleware блокирует доступ к покупке до подтверждения номера
  4. Все данные хранятся в соответствии с ФЗ-152 «О персональных данных»
Проект готов к масштабированию: замена SQLite на PostgreSQL, подключение реального SMS-шлюза, интеграция с СБП/ЮMoney и развёртывание через Docker + Nginx.
