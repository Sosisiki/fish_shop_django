О проекте

Учебный проект: Данный сайт создан в рамках дипломной работы и демонстрирует возможности интеграции AI-консультанта в e-commerce платформы. Реальные заказы не обрабатываются, доставка не осуществляется.

Концепция

«Магазин Рыбок» — учебная e-commerce платформа с AI-консультантом. Проект решает проблему несовместимости видов и нарушения биобаланса в аквариумах. ИИ анализирует параметры аквариума, подбирает совместимые виды и предупреждает о конфликтах до покупки.

Технологический стек

Backend: Django 6.0.4 (Python 3.14), архитектура MTV
Database: SQLite (с поддержкой транзакций и ORM)
AI-агент: n8n + DeepSeek V3.2 через AiTunnel
Frontend: Bootstrap 5, Vanilla JS, Fetch API
Карты: Yandex Maps JS API (геокодирование)
Безопасность: CSRF-защита, верификация email, валидация форм

Каталог и управление остатками

В базе более 50 видов пресноводных рыб, аквариумов, кормов и аксессуаров. Каждая карточка содержит:

  • Рекомендуемый объём аквариума (л)
  • Уровень сложности (Новичок / Средний / Профи)
  • Характер и совместимость с другими видами
  • Остатки на складе с блокировкой при stock ≤ 0

Система автоматически сортирует каталог: сначала доступные товары, затем «Распродано» внизу страницы.

Безопасность и верификация

Для защиты от ботов и обеспечения прозрачности сделок реализована обязательная верификация email:

  1. Ввод email, логина и пароля при регистрации
  2. Генерация 6-значного кода подтверждения с ограничением в 10 минут
  3. Проверка кода перед активацией аккаунта
  4. Все данные хранятся в соответствии с ФЗ-152 «О персональных данных»

AI-консультант

Интеграция с n8n-оркестратором и языковой моделью DeepSeek V3.2 позволяет:

  • Отвечать на вопросы о товарах на естественном языке
  • Подбирать совместимые виды рыб с учётом объёма аквариума
  • Рекомендовать оборудование и корма
  • Предупреждать о конфликтах между видами

AI-агент работает в тестовом режиме и не оформляет заказы напрямую.

Потенциал масштабирования: Архитектура проекта готова к продакшен-развёртыванию. Возможна замена SQLite на PostgreSQL, подключение реального SMS-шлюза, интеграция с платёжными системами (СБП, ЮMoney) и развёртывание через Docker + Nginx.
